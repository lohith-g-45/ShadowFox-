# 🚗 Car Price Prediction using Machine Learning

## Overview
Machine Learning project to predict used car prices using regression models.

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Workflow
Data Cleaning → Feature Engineering → Model Training → Evaluation → Model Saving

## Result
Random Forest achieved the best prediction accuracy.

## Technologies
Python, Pandas, Scikit-learn, Matplotlib
